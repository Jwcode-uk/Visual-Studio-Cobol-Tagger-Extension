## Usage ğŸ› ï¸

1. Use the hotkey `Ctrl + 1` to specify your custom tag.
2. Select one or more lines and press `Ctrl + 2` to add your custom tag to the selected lines.
3. Comment out code while keeping your tag intact with `Ctrl + 3`.
